
# University Mobile App

Mobile application developed with Flutter that connects students, parents and teachers to facilitate academic follow-up.

## Description

This application allows different users (students, parents and teachers) to access and manage academic information such as grades, schedules and personal data.

The system uses multi-profile authentication to provide different functionalities depending on the user role.

## Features

- Multi-profile authentication (Student, Parent, Teacher)
- Grade consultation and management
- Schedule access
- Personal information management
- Academic follow-up for parents
- Local data persistence using SQLite
- Data synchronization via REST API
- Modern user interface developed with Flutter

## Technologies

- Flutter
- Dart
- SQLite
- REST API
